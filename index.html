CSS

Selección de Elementos

Comprender cómo seleccionar elementos HTML utilizando selectores de CSS, como selecciones por etiqueta, clase e ID.

Propiedades Fundamentales

Conocer y ser capaz de utilizar las propiedades CSS fundamentales, como color, font-size, margin, padding, etc.

Box Model

Comprender el modelo de caja en CSS, que incluye el contenido, el padding, el borde y el margen.

Flexbox y Grid Layout

Familiarizarte con los sistemas de diseño Flexbox y Grid Layout para crear diseños web flexibles y alineados.

Posicionamiento

Saber cómo posicionar elementos utilizando propiedades como position, float, display, top, right, bottom, y left.

Estilos de Texto

Establecer estilos de texto, como font-family, font-weight, line-height, text-align, text-transform, y text-decoration.

Pseudoclases y Pseudoelementos

Utilizar pseudoclases como :hover, :active, :focus, y pseudoelementos como ::before y ::after para aplicar estilos específicos a ciertos estados o partes de elementos.

Transiciones y Animaciones

Crear transiciones suaves entre estados de elementos y animaciones usando propiedades como transition y @keyframes.

Responsive Web Design (Diseño Web Responsivo)

Diseñar páginas web que se adapten y funcionen bien en dispositivos de diferentes tamaños y resoluciones utilizando media queries.

Unidades y Valores

Comprender y utilizar diferentes unidades y valores en CSS, como píxeles (px), porcentajes (%), em (em), rem (rem), y viewport units (vw y vh).

Gestión de Colores

Definir colores utilizando nombres, códigos hexadecimales, RGB, RGBA y HSL, y aplicar gradientes de fondo.

Transformaciones

Utilizar transformaciones CSS para aplicar rotación, escala, traslación y deformación a elementos.

Fuentes Web (Web Fonts)

Incorporar fuentes personalizadas en tu diseño utilizando @font-face o servicios de fuentes web como Google Fonts.

Bordes y Sombreado

Aplicar bordes y sombreado a elementos utilizando propiedades como border, box-shadow, y text-shadow.

Estilización de Formularios

Personalizar la apariencia de formularios HTML con CSS.

Principios de Diseño y Usabilidad

Tener conocimientos básicos de principios de diseño y usabilidad para crear interfaces de usuario atractivas y funcionales.

Compatibilidad con Navegadores

Comprender las diferencias de compatibilidad entre navegadores y saber cómo solucionar problemas de rendimiento y visualización.

Optimización de Estilos

Saber cómo escribir estilos CSS eficientes y organizados para facilitar el mantenimiento y la escalabilidad de proyectos.
